Detailed Description

template<typename T>
class TiffImage< T >

Class for manipulations with monochrome TIFF images of different data types.

Work with TIFF files is based on the TinyTIFF library, so this class supports reading and writing TIFF images with the following data types: int (8, 16, 32 bits), unsigned int (8, 16, 32 bits), float (32 bits).
